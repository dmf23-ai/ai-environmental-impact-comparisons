# AI Environmental Impact Comparisons

A small, sourced website that puts AI's environmental footprint next to other large uses of water and electricity — driving, gaming, golf courses, lawns, streaming — so that arguments about it can at least start from the same numbers.

Built with [Astro](https://astro.build). Deployed on Vercel. Sources are checked weekly by a GitHub Action and stamped with a "last verified" date.

## Local development

```
npm install
npm run dev
```

Open http://localhost:4321 in a browser.

## Build

```
npm run build
npm run preview
```
